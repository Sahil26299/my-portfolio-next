### This repository contains the source code for my personal portfolio website:  
**A clean, fast, responsive portfolio built using Next.js, TypeScript and TailwindCSS for Frontend, Next Js API routes for Backend and Langchain and Groq for handling chat section**, showcasing my work, experience, skills, and projects as a full-stack / frontend / AI engineer.

Live Website: **https://my-portfolio-next-mauve.vercel.app**

---

## üöÄ Tech Stack

### **Frontend**
- Next.js 14 (App Router)
- React.js
- TypeScript
- TailwindCSS
- Framer Motion (animations)
- ShadCN / Custom UI components

### **Backend / Integrations**
- Next.js Server Actions / API Routes
- Deployed on **Vercel**

### **AI**
- LangChain.js for handling API-based workflows, using Groq API as the LLM provider.
